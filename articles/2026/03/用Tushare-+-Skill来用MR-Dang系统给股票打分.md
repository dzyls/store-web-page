有一个认真的课代表总结了一篇相当结构化的MR Dang的评估系统 [MR Dang投资体系-选股打分表，通过分数决定股票是否值得买入](https://zhuanlan.zhihu.com/p/2020082934112077680)

一眼看去，这不是天生的写成Skill给Agent来跑的素材么？前几天刚得到了[Tushare数据](https://link.zhihu.com/?target=https%3A//tushare.pro/) 的2000积分，再加上[Claude Code](https://zhida.zhihu.com/search?content_id=272147698&content_type=Article&match_order=1&q=Claude+Code&zhida_source=entity)本身就可以来写Skill，这就是做菜的原料、调料和厨师都到齐了，只差开煤气了。

所以三下五除二的，基于课代表的内容，再给加上了用Tushare去取标的财务数据和股价数据，用Tavily去网上搜标的相关的新闻，然后按着课代表给的各个维度打分，一个Skill就有了。

（有一说一，Tushare相当好用，数据全速度快。Baostock也不错，优在免费，差在数据少一些）

Skill请自取：[https://github.com/icrefin/skills](https://link.zhihu.com/?target=https%3A//github.com/icrefin/skills)

当然，Python环境和安装包要自己准备，Tushare Token和Tavily API Key要自己去获取。然后把这两个放在环境变量里供scipts去取用。

```
export TUSHARE_TOKEN=your-token
export TAVILY_API_KEY=your-api-key
```
